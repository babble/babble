Welcome to Jython 2.5a3
=======================

This is the third alpha of the 2.5 version of Jython. It
contains most of the new features for the 2.5 release.

This release fixes an installation bug on windows and restores standalone
mode.

The release was compiled on Mac OS X with JDK 5 and requires JDK 5 to run.

As an alpha release, this release is incomplete and contains bugs.  Do not
use in a production environment.
